ANIMATIONS

An animation lets an element gradually change from one style to another.

To use CSS animation, you must first specify some keyframes for the animation. Keyframes hold what styles the element will have at certain times.

ANIMATIONS AND TRANSITIONS RELATIONSHIP

  • Transitions don’t change properties
  • Transitions define the change only between property values and not the values themselves
  • Animations can change property values inside each keyframe
  • Transitions can’t loop
Not all CSS properties can be animated!
